# flutter-common-widgets-app

### 使用背景
* 鉴于目前flutter官方庞大的小部件（widget）系统的api文档，只有文字描述，而没有可视化实例。
* 我们开发这套app,可以系统的看到常用小部件（widget）的用法。
* 辅助初学者更快上手,flutter官方小部件（widget）
